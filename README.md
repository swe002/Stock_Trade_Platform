# Stock Trading Platform (Java)

## 📌 Description
This project simulates a basic stock trading environment where users can buy and sell stocks, view market data, and track their portfolio performance.

## 🚀 Features
- View real-time simulated stock market prices
- Buy and sell stocks
- Portfolio management
- Transaction history tracking
- Dynamic price updates

## 🛠️ Technologies Used
- Java
- OOP Concepts
- ArrayList
- HashMap

## ▶️ How to Run
1. Compile:
   javac Main.java
2. Run:
   java Main
